A H1 é o titulo principal do código

A h2 é o subtítulo do titulo

A h3 é um elemento de cabeçalho utilizado para definir subtítulos de terceiro nível, subdividindo tópicos dentro de uma seção

O H4 representa um cabeçalho de quarto nível, utilizado para subtítulos de seções dentro de um H3

A H5 ajuda a organizar o conteúdo, melhorando a acessibilidade e o SEO.
O H6 representa o sexto e último nível de cabeçalho (título) de uma página web, indicando a hierarquia mais baixa de importância
O form cria um formulário
O a cria um hyperlink O strong cria um texto em negrito O em cria um texto em itálico
O address cria um endereço
O area é usado para definir regiões clicáveis do hotspot dentro da imagem O title cria um nome para a página O data é usado para assosiar um conteúdo de texto legível por humanos Embed é usado para incorporar conteúdos externos bdo é utilizado para definir ou forçar a direção da leitura
